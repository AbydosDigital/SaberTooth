module.exports

App will setup PIXI renderer and root widget automatically. Just call App.update() from your render loop

new module.exports(name, options)

Parameters:
Name Type Default Description
name String My App

the name of the app
options Object

Optional parameters

Properties
Name Type Argument Default Description

Theme used by the app - by default creates a new Theme with default settings.
transparentBkg Boolean <optional>
 false

True makes background transparent
antialiasing Boolean <optional>
 false

Sets antialiasing
forceFXAA Boolean <optional>
 false

Less quality antialiasing but performs better
resolution Number <optional>
 1

Pixi renderer resolution
width Number <optional>
 800

Width of app renderer
height Number <optional>
 600

Height of app renderer
autoResize Boolean <optional>
 true

True makes app resize with browser automatically
